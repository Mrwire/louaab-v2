ok on passe# Checklist validation admin (stock & commandes)

- [ ] Confirmer une commande (1 unité) : badge stock admin passe à 0 dans les 4s, modal stock affiche 0, fiche produit “Rupture de stock”.
- [ ] Annuler ou reset une commande confirmée : badge repasse à la valeur initiale, fiche produit “En stock”.
- [ ] Restitution d’une commande livrée : badge +1 (dispo/stock), fiche produit repasse “En stock” en <10s.
- [ ] Modification manuelle du stock (modal) : badge met à jour immédiatement + front en 5s.
- [ ] Filtre/recherche inventaire : affiche la bonne valeur de stock pour le jouet filtré.
